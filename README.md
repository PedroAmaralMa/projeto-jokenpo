# projeto-jokenpo
Projeto simples de aplicativo mobile de Jokenpô, onde o usuário escolhe entre pedra, papel ou tesoura para jogar contra o computador. A escolha do “PC” é feita de forma aleatória, e o sistema define o vencedor de cada rodada. A partida continua até que um dos jogadores atinja 5 pontos; ao alcançar esse limite, o jogo é encerrado e não permite novas jogadas.
